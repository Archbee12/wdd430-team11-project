## Team Project

Handcrafted Haven is a full-stack web application that connects users with artisans and handmade products. It allows users to browse products, view artisan profiles, and manage purchases in a seamless and responsive interface.

# Features
User authentication (register & login)
Browse products with pagination
View product details
Artisan profiles and listings
Add to cart (persistent with localStorage)
Image uploads via Cloudinary
Responsive UI across devices

# Tech Stack
Frontend: React, Next.js (App Router), TypeScript
Styling: CSS / Tailwind (if used)
Backend: Next.js API routes
Database: PostgreSQL (Neon)
Authentication: bcryptjs, cookies
Validation: Zod
Media Storage: Cloudinary
HTTP / Data Layer: pg, postgres
Package Manager: pnpm

# Setup
Install dependencies
pnpm install

## Run development server
pnpm dev

## Build for production
pnpm build
pnpm start
