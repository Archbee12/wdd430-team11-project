## 📌 Overview

Handcrafted Haven is a full-stack web application that connects users with artisans and handmade products. Users can browse products, view artisan profiles, and manage cart items in a responsive and user-friendly interface.

---

## ✨ Features

- User authentication (register & login)
- Browse products with pagination
- Product details pages
- Artisan profiles and listings
- Add to cart with persistent storage
- Image uploads via Cloudinary
- Responsive design for all devices

---

## 🛠️ Tech Stack

- **Frontend:** Next.js, React, TypeScript  
- **Backend:** Next.js API routes  
- **Database:** PostgreSQL (Neon)  
- **Authentication:** bcryptjs, cookies  
- **Validation:** Zod  
- **Media Storage:** Cloudinary  
- **HTTP/Data:** pg, postgres  
- **Package Manager:** pnpm
